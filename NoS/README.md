# NoS (Network Operating System)

> "Systems Online. Connection Established."

**NoS** is a retro-styled CLI utility designed for quick network diagnostics and maintenance. Wrapped in a classic "Award BIOS" interface, it provides essential connectivity tools without the need to navigate through Windows Settings.

![NoS Banner](https://via.placeholder.com/800x400/000000/ffffff?text=NoS+System+Utility)

## ‚ö° Features

* **Flush DNS:** Clears the resolver cache to fix "Page Cannot Be Displayed" errors.
* **Renew IP:** Refreshes your local IP address to resolve connection dropouts.
* **Ping Test:** Checks latency and stability against Google DNS (8.8.8.8).
* **Speed Test:** Launches Fast.com in your default browser for a quick bandwidth check.
* **Public IP:** Retrieves your external IP address (visible to the internet).
* **Retro UI:** Classic BIOS (Color 70) aesthetic with split-pane design.
